# Airline Performance Analysis — Power BI

## Overview
Interactive Power BI dashboard analyzing US airline 
flight performance, delays, and cancellations 
for Jan–Mar 2015.

## Dataset
- Source: Kaggle (BTS 2015 Flight Data)
- Records: ~1M flights
- Tables: flights, airlines, airports

## Tools Used
- Power BI Desktop
- Power Query (data cleaning)
- DAX (measures)

## Dashboard Pages
1. Executive Overview — KPIs, trends, airline comparison
2. Delay Analysis — delay causes, heatmap, scatter
3. Route Analysis — airport map, top routes
4. Cancellations — reasons, trends by airline

## Key Metrics
- Total Flights: 1M
- OTP Rate: 79%
- Avg Arrival Delay: 7.61 mins
- Cancellation Rate: 4%
